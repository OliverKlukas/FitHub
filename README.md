# Technical Documentation
Please install all prerequisites and always run frontend and backend simultaneously.

## Project Structure

## Prerequisites
- Node: v16.x
- React: v18.x
- MongoDB: v4.x

## Run commands frontend
```bash
    cd frontend
    npm install
    npm start
 ```

## Run commands backend
```bash
    cd backend
    npm install
    mongod --dbpath "path/to/database"
    node index.js
```
